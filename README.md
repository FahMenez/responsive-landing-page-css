<h1>Easy Shopping Via Mobile</h1>
<h2>É um projeto de uma landing page responsiva feita para praticar CSS</h2>
<h3>Esse projeto foi feito com o objetivo de melhorar as minhas habilidades em CSS e responsividade. Isso foi possível graças as orientações do <a href="https://github.com/rodolfomori" target="_blank">Rodolfo Mori</a> da DevClub.</h3>
<br>
<h3>Tecnologias usadas</h3>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Google Fonts (Oswald)</li>
</ul>
<br>
<h3>Funcionalidades</h3>
<ul>
  <li>Layout responsivo com media queries</li>
  <li>Design adaptado para desktop e mobile</li>
</ul>
